# The Kingdom of Keperleys

Text-based game "The Kingdom of Keperleys" (Russian: "Королевство Кеперлейс") by Gleb Gadyatsky (Russian: Глеб Гадяцкий), a.k.a. Logger.

The game is in Russian, developed for RipURQ, and adapted for [UrqW](https://github.com/urqw/UrqW) text-based game engine.

This repository contains the source code and other resources of the game. The game is available for launch in the [UrqW engine online catalog](https://urqw.github.io/UrqW/#keperleys).

The game was first published on January 22, 2010.

Interactive fiction identifier (IFID) of the game which is its MD5 hash: B532D767EC46DA0EE4DA2A17DFCB1A8B

## Description

Тёмный чародей Мнгах угрожает королевству Кеперлейс. Его не смогли остановить самые сильные маги. Но надежда ещё есть. Нужен отважный герой, который поможет жителям многострадального королевства обрести покой и навсегда избавиться от Мнгаха. Ты готов выступить в роли такого героя? Тогда - в путь! На твоём пути Гремящие скалы и логово дракона, орды гоблинов и пираты. Но если ты умён и храбр, если не боишься оборотней и извержения вулкана, то спеши! Тучи над Кеперлейсом всё сгущаются...
